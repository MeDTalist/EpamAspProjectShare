Для того что бы сайт работал нормально, необходимо:
1)Создать базу данных
2)Выполнить в ней скрипты из файла "DataBaseGenerator.sql" Это сгенерирует структуру базы данных
3)Выполнить в базе скрипты из файла "Хранимые процедуры.sql" Это создаст необходимые для работы сайта хранимые процедуры
4)Прописать в вебконфиге ConnectioString
5)Настроить в веб конфиге SMTP клиента

Что бы сделать какого либо зарегистрированного пользователя админом,
необходимо выполнить в базе процедуру MadeAdminUser 
И передать в качестве параметра логин пользователя, например:
exec MadeAdminUser 'Admin'
С сайта такой возможности нет.
