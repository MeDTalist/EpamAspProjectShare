Для того что бы сайт работал нормально, необходимо:
1)Создать базу данных
2)Выполнить в ней скрипты из файла "DataBaseGenerator.txt" Это сгенерирует структуру базы данных
3)Выполнить в базе скрипты из файла "Хранимые процедуры.txt" Это создаст необходимые для работы сайта хранимые процедуры
4)Прописать в вебконфиге ConnectioString
5)Настроить в веб конфиге SMTP клиента
